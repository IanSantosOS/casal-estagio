Requisitos:

  1. Escreva um trecho de código HTML que cria uma tabela com duas colunas e duas linhas. Na primeira coluna da primeira linha, escreva "Nome" e na segunda coluna da primeira linha, escreva "Idade". Na primeira coluna da segunda linha, escreva "João" e na segunda coluna da segunda linha, escreva "25".
  2. Escreva um trecho de código CSS que define um estilo para um elemento <button> que tenha uma cor de fundo azul, texto branco, borda arredondada e um espaçamento interno de 10 pixels.
  3. Escreva um trecho de código CSS que selecione todos os elementos <p> dentro de um elemento com a classe "container" e defina uma margem superior de 20 pixels.
  4. Escreva um trecho de código JavaScript que adicionou um evento de clique a todos os botões da página e chame uma função chamada "handleClick" quando um botão for clicado.
  5. Escreva um trecho de código HTML que cria um formulário com um campo de entrada de texto e um botão de envio. Quando o botão for clicado, uma função chamada "submitForm" será chamada.
  6. Considere o seguinte cenário: você está desenvolvendo uma aplicação web para uma loja de livros. A aplicação deve exibir uma lista de livros disponíveis, permitir que o usuário adicione livros ao carrinho de compras e atualizar dinamicamente o total de itens no carrinho. Escreva um trecho de código HTML, CSS e JavaScript que implemente essa funcionalidade (Anexar pagina HTML ao email).
  7. Na resposta a esse e-mail, crie um formulário de login HTML e estilize-o com CSS, esse formulário ira receber do usuário e-mail e senha e posteriormente salve essas informações em uma variável Javascript ao clicar no botão para fazer login.
